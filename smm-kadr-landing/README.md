# SMM_KADR — информационный лендинг

Профессиональный адаптивный сайт на Next.js 16, React 19, TypeScript, Tailwind CSS 4 и shadcn-структуре компонентов.

## Быстрый запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000`.

## Готовая production-сборка

Папка `out` уже находится в архиве. Её можно загрузить на обычный статический хостинг.

Для локального просмотра готовой сборки:

```bash
npm install
npm start
```

## Повторная сборка

```bash
npm run typecheck
npm run build
npm start
```

## Где менять контент

Основной текст, команда, отзывы, клиенты, этапы и цены находятся в:

```text
src/components/landing-page.tsx
```

- `team` — 7 карточек команды. Замените имена, роли, описания и ссылки на фото.
- `reviews` — отзывы. Сейчас они специально помечены как черновики для согласования; перед публикацией замените их на реальные подтверждённые отзывы.
- `clientNames` — названия кейсов и партнёров.
- `prices` — пакеты и стоимость.
- `phoneDisplay`, `whatsappUrl`, `instagramUrl` — контакты.

## 3D-модель

Компонент находится в:

```text
src/components/ui/splite.tsx
```

Ссылка на Spline-сцену используется в `HeroScene` внутри `landing-page.tsx`.
Чтобы поставить другую модель робота с профессиональной камерой, замените значение `scene="..."`.
Интерактивность направления камеры за курсором настраивается внутри самой Spline-сцены.

## Изображения

Используются временные фотографии Unsplash. Для финального сайта замените их на реальные фотографии команды, офиса, съёмок и проектов SMM_KADR.

## Адаптивность

Макеты предусмотрены для 375px, 768px, 1024px и 1440px. Есть мобильное меню, адаптивные сетки, интерактивные карточки команды и карусель отзывов.
